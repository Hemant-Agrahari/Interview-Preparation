Ans. String match is a method that searches a string for a match against a regular expression and returns the matches as an array or null. example: const str = "Hello"; console.log(str.match("l")); // l console.log(str.match("z")); // null console.log(str.match(/l/g)); // [l, l] console.log(str.match(/z/g)); // null 13.What is String replace?
